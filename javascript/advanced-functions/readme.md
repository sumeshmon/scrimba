# closure
A closure is created when a function remembers and accesses variables from its outer scope, even after the outer function has finished executing.

# What is Recursion?
Recursion is when a function calls itself until a condition is met.

# What is Currying?
Currying is a technique where a function that takes multiple arguments is transformed into a series of functions, each taking one argument.
